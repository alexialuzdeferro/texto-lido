## 🌹 Sobre o Projeto
O **Vortex Editorial** é uma ferramenta autoral desenvolvida para transmutar o silêncio da escrita em manifestação sonora e estrutural. Criado sob a estética *Gothic-Abyssal*, o projeto combina a **Web Speech API** para narração feminina com um lapidador de **Markdown** automatizado.

Este é um espaço de código aberto para quem busca autonomia na criação de matérias, manifestos e ritos literários, garantindo que cada texto carregue a assinatura e a força da alma.

### ⚡ Funções Principais:
- **Invocação de Voz:** Leitura de textos com cadência ritualística e tom feminino.
- **Toque de Mestre:** Conversão instantânea para Markdown com cabeçalho editorial e assinatura personalizada.
- **Mobile First & Glassmorphism:** Design otimizado para polegares e interface translúcida.
- **Haptic Feedback:** Experiência tátil imersiva a cada interação.
